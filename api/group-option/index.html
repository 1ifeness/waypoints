group option

Default

'default'

Possible Values

Any string.

The group option identifies which group the waypoint belongs to. In most cases you will not want or need to set this option, and all waypoints will be part of the 'default' group. Setting this option to any other string will create custom groups.

Group functionality is covered in detail on the Group class page, so head over there for more information.
